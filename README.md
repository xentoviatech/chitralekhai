# Chitralekh.ai

**AI-powered document intelligence demo by [Xentovia.ai](https://xentovia.ai)**

Upload any document — invoices, contracts, IDs, handwritten notes, land records — and get structured data extracted instantly using Vision Language Models.

**Try it:** [chitralekhai.xentovia.ai](https://chitralekhai.xentovia.ai)

## Features

- Drag-and-drop document upload (PNG, JPEG, WebP)
- Real-time AI extraction with structured JSON output
- Field-level confidence scoring
- Table detection and extraction
- Copy/download results as JSON
- AI chatbot for questions and demo requests
- Enterprise-grade: documents are never stored

## Supported Document Types

Invoices, receipts, contracts, letters, forms, ID cards, medical records, land records, handwritten notes, tables, and more.

## Tech

- Frontend: Vanilla HTML/CSS/JS (this repo)
- AI: Claude Vision by Anthropic
- No build step required
